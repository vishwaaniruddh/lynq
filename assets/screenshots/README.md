# PWA Screenshots

This directory contains screenshots for the ADV Clarity Management System PWA. These screenshots are used in the web app manifest and app store listings.

## Screenshot Requirements

### Wide Form Factor (Desktop) - 1280x720
- **desktop-dashboard-wide.png**: Main dashboard showing statistics, charts, and navigation
- **desktop-inventory-wide.png**: Inventory management interface with tables and controls

### Narrow Form Factor (Mobile) - 390x844
- **mobile-dashboard-narrow.png**: Mobile-optimized dashboard with responsive layout
- **mobile-installation-narrow.png**: Installation tracking interface for mobile devices

## Screenshot Guidelines

1. **Quality**: Use high-resolution PNG format with clear, readable text
2. **Content**: Show actual application content, not placeholder data
3. **Branding**: Ensure ADV Clarity branding is visible and consistent
4. **UI State**: Show the application in a clean, professional state
5. **Accessibility**: Ensure good contrast and readable font sizes
6. **Localization**: Use English language for all text content

## File Optimization

- Compress PNG files to reduce size while maintaining quality
- Target file sizes under 1MB per screenshot
- Use tools like TinyPNG or ImageOptim for compression
- Ensure proper alt text and descriptions in the manifest

## Implementation Notes

The current files are placeholders and should be replaced with actual screenshots during implementation. Use browser developer tools or screenshot tools to capture the exact dimensions required.
